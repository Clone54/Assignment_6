# DigiTools - Professional Career Suite

DigiTools is a comprehensive suite of professional digital tools designed to empower job seekers and career-oriented individuals. From ATS-optimized resumes to executive masterclasses, DigiTools provides the edge needed in today's competitive job market.

## 🚀 Technologies Used

- **React.js**: For building a dynamic and responsive user interface.
- **Tailwind CSS & DaisyUI**: For modern, utility-first styling and pre-built accessible components.
- **JavaScript (ES6+)**: Core logic and state management.
- **Lucide React**: For beautiful, consistent iconography.
- **React-Toastify**: For real-time user feedback and notifications.
- **Framer Motion**: For smooth animations and transitions.

## ✨ Key Features

1.  **Professional Toolkit Explorer**: A curated selection of digital tools with detailed feature lists, pricing, and tags (Best Seller, Popular, etc.) displayed in a responsive 3-column grid.
2.  **Real-time Cart Management**: Seamlessly add or remove tools from your arsenal. The persistent cart tracks your total investment and updates the global navigation count instantly.
3.  **Interactive Career Roadmap**: A visual "How it Works" section that guides users through the process of choosing a plan, importing data, and exporting professional results.

## 🛠️ How to Run Locally

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.
4.  Open `http://localhost:3000` in your browser.
