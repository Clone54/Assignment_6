Project Name: DigiTools

Description: Digitools is a platform from where someone can buy digital tools like AI, and other digital services.

Used Tachnologies:
		React.js
		Tailwind CSS, DaisyUI
		JavaScript (ES6+)
		React-Toastify (NPM Package)
		JSON (for product data)

Features: 
	1. Add product to cart
  2. Remove product from cart
  3. Cart count update on navbar
